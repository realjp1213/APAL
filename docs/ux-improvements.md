## Structured input per stage (deferred to Phase 7+)

Instead of a single free-text box for all stages, consider:
- situation: separate small fields for what/where/who/doing
- avoidance: pre-filled chip options + custom add button  
- thought_input: individual thought boxes with + to add more (max 3)
- evidence_for/against: individual evidence boxes with + to add more

This reduces friction and matches patterns used in leading CBT apps.
Do not build until Phase 7 minimum — after Supabase and real user 
feedback confirms this is the right direction.
